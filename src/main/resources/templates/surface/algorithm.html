Далі сформуємо кубічне рівняння на основі значень I
λ³ - I₁λ² + I₂λ - I₃ = 0
Після цього нам потрібно знайти корені λ.Тут можна дізнатися як розв'язуються кубічні рівняння

Якщо I₃ != 0
Ми маємо три дійсні ненульові корені λ, та наше канонічне рівняння матиме вигляд: λ₁x² + λ₂y² + λ₃z² + I₄ / I₃ = 0

Якщо I₃ == 0 && I₂ != 0 && I₄ != 0
У нас знову три дійсні ненульові корені λ, але канонічне рівняння має наступний вигляд: λ₁x² + λ₂y² + 2z * (-I₄ / I₂)^(1/2) = 0

Якщо I₃ == 0 && I₂ != 0 && I₄ == 0
У цьому випадку ми три дійсні корені λ, але один з них дорівнює нулю. Отже, вигляд канонічного рівняння: λ₁x² + λ₂y² + K₃ / I₂ = 0

Якщо I₃ == 0 && I₂ == 0 && I₄ == 0 && I₁ != 0 && K₃ != 0
Маємо всього один дійсний ненульовий корінь λ. Канонічне рівняння: λ₁x² + 2y * (-K₃ / I₁)^(1/2) = 0

У всіх інших випадках маємо звичайну площину, яка не є поверхнею другого порядку
